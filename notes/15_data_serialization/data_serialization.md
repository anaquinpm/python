# Data Serialization

*   **File Formats and Persistence:** Saving and loading structured data.
*   **The `json` Module:** Using the JSON encoder and decoder to store structured data.
*   **The `csv` Module:** Reading and writing comma-separated values files.
*   **The `pickle` Module:** Serializing and persisting native Python objects.
*   *Note on External Components:* **`yaml`** is a serialization format supported through third-party packages available on PyPI *(we suggest exploring the `PyYAML` library independently)*.
