# Standard Library Overview

*   **Brief Tour of the Standard Library:** Exploring built-in modules to solve everyday problems.
*   **Interacting with the OS:** Using the `os` module for operating system interfaces.
*   **System-specific Parameters and Functions:** Using the `sys` module and runtime services.
*   **Dates and Times:** Managing and operating with the core `datetime` data type.
*   **Container Data Types:** Advanced data structures with the `collections` module.
*   **Mathematics:** Standard mathematical and numerical functions using `math`.
