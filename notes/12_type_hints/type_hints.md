# Type Hints and Annotations

*   **Type Annotations:** Using built-in types (such as generic aliases and unions).
*   **The `typing` Module:** Deep support for type hints in tooling development.
*   *Note on External Components:* **PEP 484** is the document that introduced these formal foundations and **`mypy`** is an external static type checker *(this conceptual information must be verified independently)*.
