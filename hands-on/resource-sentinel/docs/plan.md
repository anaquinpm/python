# Project 6: Server Resource Sentinel - Learning Plan

This project implements a Linux Server Resource Monitor. Development is performed 100% inside Docker containers, starting from establishing the environment, incrementally adding Python concepts, and moving towards a production-ready application.

## Project Stages

1. **[Stage 1: Dockerized Dev Environment](stage_1_docker_setup.md)**: Set up the container and volumes.
2. **[Stage 2: Basic File I/O & Data Types](stage_2_io.md)**: Variables, strings, lists, and reading virtual files.
3. **[Stage 3: Functions, Dicts & Error Handling](stage_3_functions.md)**: Functions, dictionaries, and exception handling.
4. **[Stage 4: System Commands & Regex](stage_4_subprocess.md)**: Subprocess execution and regular expressions.
5. **[Stage 5: Object-Oriented Design & Modules](stage_5_oop.md)**: Classes, object-oriented structure, and modularity.
6. **[Stage 6: Configuration & Structured Logging](stage_6_logging.md)**: Data serialization (JSON config) and Python's `logging` module.
